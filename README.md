# SnapBill – AI Powered Retail & Commerce Intelligence Platform

## Overview

SnapBill is an AI-powered retail billing and commerce intelligence platform built for Indian retail and wholesale businesses. It enables faster billing, inventory control, GST compliance, and business analytics through a hybrid online-offline architecture.

The platform includes AI voice billing, frequent offline billing, inventory management, analytics integrated with history, B2B commerce (Phase 2), and digital prescription mode for healthcare (Phase 3).

## Architecture

SnapBill follows a Clean Architecture approach:

- Presentation Layer (Flutter UI)  
- Domain Layer (State Management)  
- Data Layer (API and Local Storage)  
- Backend Service Layer (FastAPI)  
- AI Processing Layer (Gemini)  
- Cloud Database (Supabase)
